Add to WooCommerce SMS notifications to your clients for order status changes. Also you can receive an SMS message when the shop get a new order.

== Description ==

MSGWOW SMS Notification Plugin is Specially for eCommerce Stores hosted with WooCommerce. Where users and admin both get SMS Notifications for New Order, Order Status Change, Login OTP Verification, with 20+ Notifications triggers. 

== Installation ==

* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Navigate to Plugin Settings and fill settings.
